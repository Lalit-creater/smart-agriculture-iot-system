# smart-agriculture-iot-system
**Description**

This project is an IoT-based system that monitors crop health using temperature, soil moisture, and humidity sensors. It helps farmers decide when irrigation is needed, improving water efficiency and crop productivity. Smart IoT project that monitors plant health and suggest irrigation based on environmental conditions. it also provide remote monitoring using smartphone. 

**Components Used**

* microcontroller Board - ESP32
* Temperature sensor
* Soil Moisture sensor 
* Humidity sensor 
* Connecting Wires
* IoT APP

**Working**

The system collects real-time data from sensors:
* Temperature
* Soil Moisture
* Humidity  

This data is sent to the app, where users can monitor plant conditions remotely. Based on the data, the system indicates plant health and about the need of irrigation.

**Future Improvements**

- Automatic irrigation system   
- Cloud data storage
